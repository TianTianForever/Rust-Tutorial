fn new(size: usize) -> ThreadPool

Create a new ThreadPool. The size parameter is the number of thread in pool. The ThreadPool::new use for loop counter to generate an id, creade a new Worker with that id, and store the worter in the vertor. Note that the 'new' function will panic if the size is zero.
